== Description ==

SKT Plants is a garden landscape recycle based responsive flexible easy to use lightweight WordPress theme based on Gutenberg block editor. Can be used for conservation, eco friendly, green renewable energy, non profit, NGO, ecology, gardening, earth, environment, wind, hydro, nature, agriculture, farming, flowers, organic, nursery, wildlife preservation, lawn, fertilizer, forest, bio etc.
